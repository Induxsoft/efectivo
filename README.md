# Efectivo (Gestión de efectivo).
El paquete Efectivo (Gestión de efectivo) de V12 es el módulo que permite la administración de cuentas bancarias y movimientos de efectivo, funcionando como el núcleo para controlar la liquidez de la empresa. Su propósito es centralizar la gestión de cuentas, depósitos, retiros y transferencias, integrándose con los demás módulos financieros del ERP para garantizar coherencia y trazabilidad.
